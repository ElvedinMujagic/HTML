Uređene liste

    Uređene liste koriste brojeve kao oznake. Web browser-i prikazuju arapske brojeve, kao podrazumijevanu vrijednost za oznake uređenih lista.
    Tip oznake definiše se pomoću različitih vrijednosti atributa type.
    type="a | A | i | I | 1"

    Standardna vrijednost atributa type="1"

    1. List item 1
    2. List item 2

    Primjer type="a"

    1. List item 1
    2. List item 2

    Primjer type="A"

    1. List item 1
    2. List item 2

    Primjer type type="i"

    Uređena lista sa arapskim brojevima
    1. List item 1
    2. List item 2

    Primjer type type="I"

    Uređena lista sa arapskim brojevima
    1. List item 1
    2. List item 2
